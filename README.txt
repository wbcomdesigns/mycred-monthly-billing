This plugin will add monthly auto point deduction functionality to MyCred plugin.

== Description ==

The MyCred add-on work with buddypress and third party plugins for specific hooks to auto point deduction functionality for all members.

== Installation ==

This section describes how to install the plugin and get it working.

1. Download the zip file and extract it.
2. Upload `mycred-monthly-billing` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the \'Plugins\' menu.
4. Alternatively you can use WordPress Plugin installer from Dashboard->Plugins->Add New to add this plugin
5. Enjoy
